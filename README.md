## Bistro Tiffin & Health
Team: Shadow Stack

Bistro Tiffin & Health is a web-based application designed to integrate daily meal planning with basic health tracking. The platform allows users to estimate their calorie requirements, log food intake, and receive a structured view of their daily nutrition. It also provides a simple mechanism to place tiffin orders based on nutritional needs.

## Overview

The project focuses on combining convenience with health awareness. Instead of separately tracking diet and ordering meals, users can manage both in one place. The system maintains a balance between usability and functionality, making it suitable for everyday use.

## Key Features
User login interface for basic access control
Calorie requirement calculation using standard BMR formula
Meal logging through a chatbot-style interface
Automatic update of daily nutrition metrics
Summary of calorie, protein, carbohydrate, and fat intake
Location tracking for delivery support
Option to place tiffin orders based on nutritional data
Technology Stack
Frontend: HTML, CSS, JavaScript
APIs:
Browser Geolocation API
OpenStreetMap (for map rendering)
Working

The application starts with a login screen. Once inside, users can calculate their daily calorie requirements by entering basic details such as age, height, weight, and gender.

Meals can be logged through a chat interface, which updates the nutritional report dynamically. Based on this data, users can review their daily intake and proceed to order a tiffin aligned with their dietary needs.

The system also includes a location feature to assist with delivery tracking.
